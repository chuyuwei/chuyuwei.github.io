如何快速解决动态规划的问题

动态规划是面试中的必考题目，很多同学都不太会做，博主也是这样，这次看到一篇描述非常好的方法，楼主通过这种方法解决了很多题目，这篇bolg的地址是 http://blog.csdn.net/baidu_28312631/article/details/47418773
楼主发现所有的动态规划都可以使用递归写出来，如果递归中出现重复的路径，那么将重复的记录下来，在下次不做计算就是动态规划，话不多说，直接拿例子开讲。

leetcode 198 house robber

You are a professional robber planning to rob houses along a street. Each house has a certain amount of money stashed, the only constraint stopping you from robbing each of them is that adjacent houses have security system connected and it will automatically contact the police if two adjacent houses were broken into on the same night.
Given a list of non-negative integers representing the amount of money of each house, determine the maximum amount of money you can rob tonight without alerting the police.

这题目是为了发现小偷偷盗最多钱数，有一排房子，不能连续偷两个房子，最大能偷多少钱，比如 1 7 4三个房子，就偷第二个房子，投了7块，比如1 4 7就偷第一和第三个房子，一共8块。
为了解决这道题目，我们可以从递归→有记录的递归→递推这样的方式来解决
递归 

int MaxMoney(int i, vector<int>& nums)
{
  if (i > nums.size()-1)
     return 0;
  return max( nums[i]+MaxMoney(i+2,nums),MaxMoney(i+1,nums) );
}

一般人都能很快写出这个递归表达式，但是发现超时了，这是由于做了大量重复的计算，比如计算MaxMoney（10）的时候和MaxMoney（9）的时候都计算了MaxMoney（10），以后的每一步都会计算一遍上一遍的东西，所以我们可以将重复计算的用数组记录下来，在下次使用的时候直接使用，可以使用money记录每次的最大数目，如下
有记录的递归 

//加入记录函数  
int* money;
int MaxMoney(int i, vector<int>& nums)
{    
if (i > nums.size()-1)
    return 0;
if (money[i] != -1) return money[i];
return max( nums[i]+MaxMoney(i+2,nums),MaxMoney(i+1,nums) );
}
int rob(vector<int>& nums) {
if (nums.size() == 0) return 0;
money = new int[nums.size()];
for (int i = 0; i < nums.size(); i++)
    money[i] = -1;
return MaxMoney(0,nums);
}

“矩阵行走问题

矩阵行走问题的几种处理 #（随时更新）

在做leetcode的时候发现经常会涉及到矩阵的运算，通常处理的对象是这样的 

vector<vector<int>> object;

在处理矩阵问题的时候，通常需要遍历周围一圈，分别是 

object[i-1][j-1];
object[i-1][j];
object[i][j-1];
object[i+1][j+1];
object[i][j+1];
object[i+1][j];
object[i-1][j+1];
object[i+1][j-1];

需要考虑的问题是0<i<object.size() 0<j<object.size(),一般分别做判断，做8次if判断可以解决这个问题，有些问题可以通过这样的方式解决，比如leetcode 661 image smoother可以用这样的方式解决，但是 675 Cut Off Trees for Golf Event 无法通过这样的方式解决，可以通过这样的方式判断减少代码量 

vector<vector<int>> dir = { { -1, 0 }, { 0, -1 },{ 0, 1 }, { 1, 0 } };
for (int k = 0; k<dir.size(); k++)
{
    int nextx = i + dir[k][0];
    int nexty = j + dir[k][1];
    if (nextx>-1 && nextx < grid.size() && nexty>-1 && nexty < grid[0].size())
    {
    }
}

这样比起8个if减少了很多操作

红宝书第二章状态管理和绘制几何物体

状态管理和绘制几何物体

引言：为了充分了解《OpenGL编程指南》这本书，同时也是作为日记的存在，笔者特意开次博客，希望笔者的阅读记录能够为大家也有所帮助。

glLineStipple 点画线

本章有一个比较少用的OpenGL的绘制函数是点画线模式，具体是 

glLineStipple(1,0x3F07);        
glEnable(GL_LINE_STIPPLE);

其中0x3F07是十六进制表示，二进制下是001111100000111，表示，从有右到左表示先绘制三个像素，在连续五个像素留空，在绘制6个像素，最后两个像素留空。
如下图实例程序结果

源码如下 

#include <glut.h>
#define drawOneLine(x1,y1,x2,y2) glBegin(GL_LINES);\
glVertex2f((x1),(y1)); glVertex2f((x2), (y2)); glEnd();

void init(void){
    //设置清除颜色为黑色
    glClearColor(0.0, 0.0, 0.0, 0.0);
    //着色模式，使用图元中某个个顶点颜色渲染整个图元
    glShadeModel(GL_FLAT);
}

void display(void)
{
    int i;
    //清除整个窗口为当前颜色
    glClear(GL_COLOR_BUFFER_BIT);
    //设置绘制颜色为白色
    glColor3f(1.0, 1.0, 1.0);
    //启用直线点画模式
    glEnable(GL_LINE_STIPPLE);

    //定义点画模式，当前模式是
    glLineStipple(1, 0x0101);
    drawOneLine(50, 125, 150, 125);
    glLineStipple(1, 0x00FF);
    drawOneLine(150, 125, 250, 125);
    glLineStipple(1, 0x1C47);
    drawOneLine(250, 125, 350, 125);

    //与上面不同的是设置了宽度
    glLineWidth(5.0);
    glLineStipple(1, 0x0101);
    drawOneLine(50, 100, 150, 100);
    glLineStipple(1, 0x00FF);
    drawOneLine(150, 100, 250, 100);
    glLineStipple(1, 0x1C47);
    drawOneLine(250, 100.0, 350, 100);

    glLineStipple(1, 0x1C47);
    glBegin(GL_LINE_STRIP);
    for ( i = 0; i < 7; i++)
        glVertex2f(50.0 + ((GLfloat)i * 50), 75.0);
    glEnd();

    for ( i = 0; i < 6; i++)
    {
        drawOneLine(50.0 + ((GLfloat)i * 50), 50.0,
            50.0 + ((GLfloat)(i + 1)*50.0), 50.0);
    }

    glLineStipple(5, 0x1C47);
    drawOneLine(50.0, 25.0, 350.0, 25.0);

    //关闭点画模式
    glDisable(GL_LINE_STIPPLE);
    //强制以前的opengl命令开始执行
    glFlush();
}

//w和h表示新的窗口的大小
void reshape(int w, int h)
{
    //占据整个新的窗口
    glViewport(0, 0, (GLsizei)w, (GLsizei)h);
    //对投影矩阵应用随后的矩阵操作
    glMatrixMode(GL_PROJECTION);
    //将当前坐标系原点移动到屏幕中间
    glLoadIdentity();
    //位于左下角坐标是0，0，右上角坐标是w,h
    gluOrtho2D(0.0, (GLdouble)w, 0.0, (GLdouble)h);
}
int main(int argc, char** argv)
{
    //对GLUT初始化
    glutInit(&argc,argv);
    //指定颜色模式
    glutInitDisplayMode(GLUT_SINGLE | GLUT_RGB);
    //指定窗口左上角位置
    glutInitWindowSize(400, 150);
    //指定窗口的大小
    glutInitWindowPosition(100, 100);
    //创建窗口
    glutCreateWindow(argv[0]);
    //初始化RGBA模式窗口黑色
    init();
    //调用绘制函数
    glutDisplayFunc(display);
    //窗口大小改变的回调函数
    glutReshapeFunc(reshape);
    //循环调用
    glutMainLoop();
    return 0;
}
